# PNT LSM Academy

A starter web-based learning system for PNT robotics training.

## What is included

- `index.html`: Landing page with hero, feature sections, modules, and AI tutor.
- `styles.css`: Responsive visual design with modern dark theme styling.
- `app.js`: Interactive AI chat stub, module overlay, and sample quiz/game actions.

## How to use

1. Open `index.html` in your browser.
2. Explore the learning modules section.
3. Ask questions in the AI Tutor Studio.
4. Use the quiz and game placeholders to test interaction.
